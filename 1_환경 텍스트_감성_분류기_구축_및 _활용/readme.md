# 환경 텍스트 감성 분류기 구축 및 활용

## 연구목표
- 환경 SNS 데이터 수집 감성분석, 결과 - 발신을 주기적으로 반복하는 '환경텍스트 감성 분류기 구축'
- 2018~2019년 개발한 기후변화 감성분류기를 환경 전 분야로 확장
- 감성분류 결과의 추이를 시각화 하여 확인할 수 있는 웹 기반 유저 인터페이스 제공
